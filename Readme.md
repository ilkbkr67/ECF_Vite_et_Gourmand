# Installation

Le projet utilise Symfony donc il faut tout les prérequis c'est a dire avoir : 

    - PHP 8.4 minimum
    - Composer

Verifier avec la commande si tout est ok

    - symfony check:requirements

Une fois les prérequis installé lancé la commande

    - symfony new my_project_directory --version="8.0.x" --webapp"
